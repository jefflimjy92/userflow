# 더바다 · 고객여정 운영맵 대시보드

보험 환급 서비스 "더바다"의 고객여정 운영맵을 FigJam 스타일의 인터랙티브 웹 대시보드로 구현한 프로젝트입니다.

## 여정 유형
- 💰 3년 환급 (9단계): 보상팀 TM -> 미팅 -> 청구 -> 환급
- ⚡ 간편 청구 (6단계): 온라인 셀프 청구
- 🤝 소개/가족연동 (6단계): 소개 발생 -> 가족연동 -> 전환

## 뷰 모드
- 운영맵: 카드형 단계별 노드 뷰
- 플로우차트: 베지어 곡선 트리 뷰 (엣지 선택/편집 지원)
- 업무 현황: 서비스 블루프린트 그리드

## 기술 스택
React 18 + TypeScript + Tailwind CSS v4 + Vite + react-dnd + lucide-react

## 설치 및 실행
```bash
pnpm install
pnpm dev
```

## 빌드
```bash
pnpm build
```
